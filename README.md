# Personal Portfolio Website

A clean, responsive personal portfolio website built with HTML and CSS.
Showcases my profile, education, technical skills, projects, work experience,
and certifications — deployed live via GitHub Pages.

## Live Demo

Visit the live portfolio here:
https://anjli20.github.io/portfolio

## Sections

- Profile — photo, name, and a brief introduction
- Education — academic background and qualifications
- Skills — technical and data science skills
- Projects — key projects with descriptions
- Experience — internships and work experience
- Certifications — courses and certificates completed

## Tech Stack

| Technology | Purpose |
|---|---|
| HTML | Page structure and content |
| CSS | Styling and layout |
| GitHub Pages | Free static site hosting and deployment |
